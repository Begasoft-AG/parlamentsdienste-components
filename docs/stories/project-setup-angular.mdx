# Project Setup Angular

Node Version: 22<br />
Angular Version: 19

<br />

1. Start a new angular project using the Angular CLI:

    ```bash
    ng new my-angular-app
    ```

2. Install the parlamentsdienste components from npm:

    ```bash
    cd my-angular-app
    npm install @parlamentsdienste/pdcomponents-angular
    ```

3. To add fonts and icons add this objectsto the `angular.json` assets array:

    ```json
        "assets": [
              {
                "glob": "**/*",
                "input": "node_modules/@parlamentsdienste/pdcomponents-core/dist/parlamentsdienstecore/assets",
                "output": "assets"
              },
        ]
    ```

4. To add the base styles and fonts add these two lines to the `styles` array in `angular.json`:

    ```json
        "styles": [
            "node_modules/@parlamentsdienste/pdcomponents-core/dist/parlamentsdienstecore/typography.css",
            "node_modules/@parlamentsdienste/pdcomponents-core/dist/parlamentsdienstecore/parlamentsdienstecore.css"
        ],
    ```

5. Call setAssetPath in your app-initializer and set it to the baseUrl of your project

    ```typescript
    import { setAssetPath } from '@parlamentsdienste/pdcomponents-core/components';

    bootstrapApplication(AppComponent, {
        providers: [
            provideAppInitializer(() => {
                setAssetPath('http://localhost:4200');
            }),
            provideZoneChangeDetection({ eventCoalescing: true }),
            provideRouter(appRoutes),
        ],
    }).catch(err => console.error(err));
    ```
